Data — staff & units

Edit the staff data. Changes save automatically to the data/ Markdown files (one per unit) a moment after you stop typing, with a timestamped backup each time — so a new person you add is kept even if you don't click Save. (Save & reload also refreshes the dashboard grid.) Entering a Google Scholar URL sets that person's status to set so their metrics are fetched on the next view. Scholar data (citations, h-index, sparkline) is cached separately and re-fetched via Refresh in the Data menu or Refresh Data from Google Scholar on a person's card.
